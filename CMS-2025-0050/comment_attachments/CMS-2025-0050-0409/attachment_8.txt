White Paper: Agentic AI Architecture
Powering Addison Care
Executive Summary
Addison Care represents a breakthrough in intelligent virtual health assistance. A dynamic,
modular, agentic AI architecture built on Microsoft’s open-source Semantic Kernel is at the heart
of its operation. This architecture allows Addison to be a deeply personalized, always-on care
partner capable of reasoning, adapting, and responding across various health and wellness
scenarios. Powered by large language models (LLMs), Addison orchestrates complex tasks using
streaming and non-streaming workflows, delivering scalable and proactive health engagement.
1. Semantic Kernel Foundation & Model Flexibility
The system is built upon Microsoft’s Semantic Kernel—a robust orchestration layer that enables
advanced memory management, goal fulfillment, and agent coordination. This LLM-agnostic
platform allows for integration with a range of large language models, including OpenAI, Azure
OpenAI, Anthropic, and Mistral. It’s designed for enterprise scalability, regulatory compliance, and
seamless adaptation to emerging AI advancements.
2. Intent Detection and Dynamic Agent Routing
Addison uses an Intent Detection Agent to interpret user prompts and determine appropriate task
execution. Tasks are categorized into streaming responses (real-time conversational needs) and
non-streaming responses (background or batch operations). This dual-mode functionality ensures
performance efficiency while maintaining responsive user engagement.
3. Modular Specialized Agents
The AI framework features four core agents, each managing a distinct health domain:
A. Settings Agent
- Vital Readings Analyzer
- Vitals Troubleshooter
- Vitals Reminders Customizer
B. Vitals Agent
- Manages real-time and historical vitals data
- Initiates alerts and educational interactions
C. Medication Agent
- Med Reminders Customization
- Med Activity Tracker
D. Survey Agent
- Survey Reminder Agent
- Survey Results Analyzer
4. Continuous Learning and Adaptation
Addison’s agentic AI continuously learns through user feedback and behavioral observation. This
enables it to adjust routines, optimize reminders, and personalize interventions over time. The
result is an evolving care model that supports early intervention, adherence, and holistic wellness.
